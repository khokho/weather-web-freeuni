## ინფორმაცია

* __სათაური__: საქართველოს ამინდის საიტი
* __აღწერა__: საიტზე შესაძლებელი იქნება საქართველოს ქალაქებში ამინდის გაგება
* __API__: [openweathermap API](https://openweathermap.org/api) ან Mocked API
