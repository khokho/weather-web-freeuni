# ამინდის საიტი

## ინფორმაცია

* __სათაური__: ამინდის საიტი
* __აღწერა__: საიტზე შესაძლებელია მსოფლიოს ქალაქებში ამინდების გაგება. 
* __API__: [metaweather API](https://www.metaweather.com/api/)
